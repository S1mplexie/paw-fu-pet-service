## 运行测试方法

### 方法1：使用IDE运行

1. 在IDEA中打开项目
2. 右键点击 `src/test/java` 目录
3. 选择 "Run 'All Tests'"
4. 查看测试结果

### 方法2：使用Maven命令

```bash
# 运行所有测试
mvn clean test

# 运行指定测试类
mvn test -Dtest=UserControllerTest

# 生成测试报告
mvn clean test jacoco:report
```

### 方法3：使用Maven Wrapper（如果项目已配置）

```bash
./mvnw clean test
```

## 注意事项

1. **测试隔离**：每个测试方法独立运行，使用 `@BeforeEach` 初始化测试环境
2. **Mock使用**：使用 `@MockBean` 和 `@Mock` 模拟依赖，避免真实数据库操作
3. **测试数据**：使用H2内存数据库，测试完成后自动清理
4. **断言验证**：使用JUnit 5断言，验证方法调用次数和参数
